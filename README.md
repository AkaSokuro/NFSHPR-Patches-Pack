# NFSHPR - Patches Pack
[![GitHub release](https://img.shields.io/github/v/release/AkaSokuro/NFSHPR-Patches-Pack?color=red)](https://github.com/AkaSokuro/NFSHPR-Patches-Pack/releases)

An **unofficial patch collection** that fixes several bugs for **Need for Speed: Hot Pursuit Remastered**.

## Disclaimer

This modification is not affiliated with **Criterion** or **Stellar Entertainment**. 

It is an **unofficial community patch** for the game. The source code is fully disclosed to ensure transparency and safety.

## Features
### Essentials
- `Affinity Fix` - Workaround to fix the same bug found in the 2010 version by disabling CPU 0 affinity on game launch
- `Friendlist Patch` - Fixes a bug that causes the game to crash when trying to load more than 199 friends in solo missions
- `Match End Crash Fix` - Fixes a rare crash that may occurs at the end of a match

### Miscellaneous
- `Game Priority` - Set the game process's priority on launch
- `Disable Background Video` - Disables in-game videos (**disabled by default**)

## Installations
- Download the latest release **ZIP** file from the Releases. (**Not Source code**)
- Extract all files into the game directory where `NFS11Remastered.exe` is located

## Uninstallations
- Delete `dinput8.dll` from the game directory. (**Optional**, other mods might require it)
- Delete `NFSHPR-Patches-Pack.asi` from the `scripts` folder **Or** delete the `scripts` folder.

## Compilations
**Visual Studio 2022** and **C++20** are required for project compilation.
